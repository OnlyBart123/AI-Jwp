    Laboratoria

    Element Opis Akcja
    JavaScript JavaScript to skryptowy język programowania, który pozwala na tworzenie dynamicznych treści na stronach internetowych. Jest on szeroko używany do interakcji z użytkownikami, manipulacji elementami HTML oraz obsługi zdarzeń.
    DOM DOM (Document Object Model) to interfejs programistyczny, który reprezentuje dokumenty HTML i XML jako strukturę obiektową. Dzięki DOM programiści mogą dynamicznie zmieniać zawartość, strukturę i styl dokumentu.
    NodeJS Node.js to środowisko uruchomieniowe, które pozwala na uruchamianie kodu JavaScript po stronie serwera. Dzięki temu programiści mogą tworzyć aplikacje internetowe oparte na JavaScript zarówno na froncie, jak i na backendzie.
    NodeJS zdarzenia W Node.js system zdarzeń umożliwia obsługę asynchronicznych zdarzeń w aplikacji. Programiści mogą rejestrować funkcje, które będą wywoływane, gdy dane zdarzenie wystąpi, co pozwala na efektywne zarządzanie interakcjami użytkowników.
    jQuery jQuery to biblioteka JavaScript, która upraszcza manipulację dokumentami HTML, obsługę zdarzeń oraz efekty animacji. Dzięki jQuery programiści mogą łatwiej tworzyć interaktywne i responsywne strony internetowe.
